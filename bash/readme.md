## Install

1. Environment
Create .env file

```console
touch .env
```

contexts .env file:
```console
export MOSQUITTO_USER=mosquitto
export MOSQUITTO_PASSWORD="your password"
export MOSQUITTO_PORT=1883
export MOSQUITTO_QOS=1
export MOSQUITTO_RETAINED=0
```
